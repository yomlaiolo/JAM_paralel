# DOUBLE VISION

Double Vision is a game made for the EPITECH Game JAM.
The objective is to resolve puzzles and levels while two players are at two sides of the ground.

## Requirements

`Build`
- C++20 compiler
- Makefile

`Libraries`
- SFML

## Usage
```bash
USAGE:
  ./double_vision
  
COMMANDS:
  NONE
```
